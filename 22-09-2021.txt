----------------- EXERCICIO 2

Desenvolva um programa em C que leia uma string do usuário e conte a quantidade de vogais e a quantidade de consoantes dessa string.

**Dica:** Para simplificar o exercício, você pode considerar que todo caractere que não é uma vogal é uma consoante. Se quiser realmente testar se o caracter é uma consoante, lembre-se da tabela ASCII.

**Dica 2:** Lembre-se que maiúsculas e minúsculas são diferentes em C: `'A' ? 'a'`.

**Dica 3:** Lembre-se dos operadores lógicos: E (`&&`) e OU (`||`).

Exemplo do programa executando:

#include <stdio.h>
#include <string.h>

int main()
{

    char string[100]; 
    int i, vogais=0, consoantes=0;
    
    printf("\nDigite uma string: ");
    scanf("%s", string);
    
    for(i = 0; string[i] != '\0'; i++) 
    {       
    
        if (string[i] =='a' || string[i] =='A' || string[i] =='e' || string[i] =='E' || string[i] =='i' || string[i] =='I' || string[i] =='o' || string[i] =='O' || string[i] =='u' || string[i] =='U')
        {
            vogais = vogais + 1;
            
        }
        else
        {
            consoantes = consoantes + 1;
        }
        
        
    }
    
    printf("\nA Quantiade de Vogais é: %d", vogais);
    printf("\nA Quantiade de Consoantes é: %d", consoantes);
        
    return 0;
}
